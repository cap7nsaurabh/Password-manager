# Password-manager
## Description
A user can have many passwords from various sites, having a common password for all of them is not a good idea and as passwords are growing complex now a days it is hard to remember
all the passwords so this may project may help. This is the sole motive of this project, to store the user passwords in a secure manner, The passwords are encrypted using AES and then
stored in the database.
